BINGE+ Streaming Service

Features

  • Movie Listings: Fetches movies from the backend using an API and displays them dynamically.
  • Search Functionality: Users can search for movies by title.
  • Feedback: Allows users to provide feedback about the platform.
  • Modal Alerts: Displays important notices for users.

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express
  • DevOps: Docker, GitHub Actions, CI/CD Pipelines, Terraform
  • Database: MongoDB (assumed)

Project Structure

The project structure is as follows:

  • /.github/workflows/: Contains CI/CD pipeline configurations using GitHub Actions.
  • /server: Contains the backend server logic.
  • Frontend Files: index.html, movie_detail.html, feedback.html, etc.
  • DevOps Files: Dockerfile, start_nodemon.sh, etc.

Setup Instructions

  1. Clone the Repository: git clone https://github.com/yourusername/bingeplus.git
  2. Install Dependencies: Navigate to the `/server` folder and install Node.js dependencies using npm install.
  3. Run the Application: Start the backend server with npm start.
  4. Docker: Build and run the app using Docker: docker build -t bingeplus .

API Endpoints

  • GET /movies/find: Fetches a list of available movies.
  • POST /feedback: Submits user feedback.

Future Enhancements

  • Integration with Streamlit for Python-based enhancements.
  • Expanded movie database with more content.
  • User authentication for login/signup.
